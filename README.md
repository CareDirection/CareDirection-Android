Naming Rule
* ## View id값
	* 버튼 : btn_
	* RecyclerView item.xml : rv_list_item~
	* RecyclerView : rv_~_
	* 텍스트뷰 : txt_~
	* 이미지뷰 : img_~
	* 에디트텍스트 : edit
  	
* # Border 
	* ### radius
    		* 기본 틀 : color_border_form
		* 6dp  -> color_border_square / gray_boder_line
		* 10dp -> color_border_long
		* 18dp -> white_border_fill / white_border_line
		* 23dp -> color_border
